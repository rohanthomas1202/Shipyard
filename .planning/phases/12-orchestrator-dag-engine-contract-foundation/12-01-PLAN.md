       - `__init__(self, dag_id: str = "")`: Creates `self._graph = nx.DiGraph()`, stores `self.dag_id = dag_id or _new_id()`
       - `add_task(self, task_id: str, *, label: str = "", description: str | None = None, task_type: str = "agent", contract_inputs: list[str] | None = None, contract_outputs: list[str] | None = None, metadata: dict | None = None) -> TaskNode`: Creates TaskNode, adds to graph with node data, returns it

       - `get_ready_tasks(self, completed: set[str]) -> list[str]`: Returns task IDs whose all predecessors are in completed set and are not themselves completed
       - `get_execution_waves(self) -> list[list[str]]`: Returns `[list(gen) for gen in nx.topological_generations(self._graph)]`
       - `get_ancestors(self, task_id: str) -> set[str]`: Returns `nx.ancestors(self._graph, task_id)`
       - `get_task(self, task_id: str) -> TaskNode | None`: Returns TaskNode from graph node data
       - `task_count` property: `len(self._graph.nodes)`
       - `edge_count` property: `len(self._graph.edges)`
       - `@classmethod from_definition(cls, dag_id: str, tasks: list[dict], edges: list[dict]) -> "TaskDAG"`: Factory that builds DAG from JSON-like structure (tasks = [{id, label, ...}], edges = [{from_task, to_task}])

    5. Create `tests/test_dag.py` with pytest tests covering all behaviors listed above. Use `pytest.mark.asyncio` only for async tests (these are all sync). Import `from agent.orchestrator.dag import TaskDAG`.
