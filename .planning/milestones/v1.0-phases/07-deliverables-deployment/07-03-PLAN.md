1. `heroku create shipyard-app`
2. `heroku buildpacks:add heroku/nodejs`
3. `heroku buildpacks:add heroku/python`
4. `heroku config:set OPENAI_API_KEY=... LANGCHAIN_TRACING_V2=true LANGCHAIN_API_KEY=... LANGCHAIN_PROJECT=shipyard`
5. `git push heroku main`
