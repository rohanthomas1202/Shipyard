Create `scripts/deploy_railway.py` with the following implementation per D-06 (fully automated Railway deploy).

Module docstring: `"""Automated Railway deployment for rebuilt Ship app.\n\nPer D-06: Fully automated -- provisions DB/secrets, deploys, verifies health.\nPer D-07: Secrets managed through Railway environment variables.\n"""`

Imports:
```python
import argparse
import asyncio
import logging
import sys
import time

import httpx

from agent.tools.shell import run_command_async
```

Constants:
- `logger = logging.getLogger(__name__)`
- `HEALTH_CHECK_TIMEOUT = 300` (5 minutes)
- `HEALTH_CHECK_INTERVAL = 10` (seconds)
- `DEFAULT_ENV_TEMPLATE = "scripts/railway_template.env"`

Define `async def check_auth() -> None`:
1. `result = await run_command_async(["railway", "whoami"], timeout=10)`
2. If `result["exit_code"] != 0`: raise `RuntimeError("Railway CLI not authenticated. Run 'railway login' first.")` (per Pitfall 5 and UI-SPEC copy)
3. `logger.info("Railway authenticated as: %s", result["stdout"].strip())`

Define `async def init_project(project_dir: str) -> None`:
1. `result = await run_command_async(["railway", "init"], cwd=project_dir, timeout=30)`
2. If exit_code != 0 and "already linked" not in stderr: raise RuntimeError
3. `logger.info("Railway project initialized in %s", project_dir)`

Define `async def provision_postgres(project_dir: str) -> None`:
1. `result = await run_command_async(["railway", "add", "--plugin", "postgresql"], cwd=project_dir, timeout=60)`
2. If exit_code != 0: `logger.warning("Postgres addon may already exist: %s", result["stderr"])`
3. Else: `logger.info("Postgres addon provisioned")`

Define `async def set_env_vars(project_dir: str, env_template: str) -> int`:
1. Open env_template, read lines
2. `count = 0`
3. For each non-empty, non-`#`-prefixed line: call `await run_command_async(["railway", "variable", "set", line.strip()], cwd=project_dir, timeout=15)`
4. Increment count, log each variable name (key only, not value for security)
5. Return count
6. Log: `logger.info("Set %d environment variables", count)`

Define `async def deploy(project_dir: str) -> None`:
1. `result = await run_command_async(["railway", "up", "--detach", "--ci"], cwd=project_dir, timeout=300)`
2. If exit_code != 0: raise `RuntimeError(f"Deploy failed: {result['stderr']}")`
3. `logger.info("Deploy initiated")`

Define `async def generate_domain(project_dir: str) -> str`:
1. `result = await run_command_async(["railway", "domain"], cwd=project_dir, timeout=30)`
2. `domain = result["stdout"].strip()`
3. If not domain: raise RuntimeError("Failed to generate Railway domain")
4. `logger.info("Public domain: %s", domain)`
5. Return domain

Define `async def verify_health(url: str) -> bool`:
1. `deadline = time.monotonic() + HEALTH_CHECK_TIMEOUT`
2. Loop while `time.monotonic() < deadline`:
   a. Try `async with httpx.AsyncClient() as client: resp = await client.get(f"https://{url}/health", timeout=10)`
   b. If `resp.status_code == 200`: log success, return True
   c. Except any exception: pass
   d. Log: `logger.info("Health check pending... retrying in %ds", HEALTH_CHECK_INTERVAL)`
   e. `await asyncio.sleep(HEALTH_CHECK_INTERVAL)`
3. Return False

Define `async def run_deploy(project_dir: str, env_template: str) -> str`:
1. Print: `"Deploying rebuilt Ship to Railway..."`
2. `await check_auth()`
3. `await init_project(project_dir)`
4. `await provision_postgres(project_dir)`
5. `count = await set_env_vars(project_dir, env_template)`
6. `await deploy(project_dir)`
7. `domain = await generate_domain(project_dir)`
8. Print: `f"Verifying health at https://{domain}/health ..."`
9. `healthy = await verify_health(domain)`
10. If healthy: print `f"Ship deployed successfully: https://{domain}"`, return domain
11. Else: raise `RuntimeError(f"Deployed but health check failed after 5 minutes: {domain}")` (per UI-SPEC copy)

Define `def main()` with argparse:
- `--project-dir` required, help "Path to rebuilt Ship project directory"
- `--env-template` default `DEFAULT_ENV_TEMPLATE`, help "Path to Railway env template"
- `--log-level` default "INFO", choices `["DEBUG", "INFO", "WARNING"]`

In main:
1. Parse args, configure logging
2. `domain = asyncio.run(run_deploy(args.project_dir, args.env_template))`
3. Print final domain
4. `sys.exit(0)`
5. Catch RuntimeError: print error, `sys.exit(1)`

Add `if __name__ == "__main__": main()` at bottom.
