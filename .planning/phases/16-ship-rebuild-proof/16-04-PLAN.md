   ```

     --max-concurrency 10 \
     --log-level INFO
   ```

   ```
   ls -la /tmp/ship-rebuilt/
   test -f /tmp/ship-rebuilt/package.json
   ```

6. Run npm install in the output directory to ensure dependencies are resolved:
   ```
   cd /tmp/ship-rebuilt && npm install
   ```
   This is mandatory -- the rebuild generates package.json but npm install must run before build/deploy.
