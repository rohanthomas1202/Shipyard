    1. In `agent/tracing.py`, add a standalone function (not part of TraceLogger class):

       ```python
       async def share_trace_link(run_id: str, metadata_filter: dict | None = None) -> str | None:
           """Generate a public LangSmith shared trace link for a run.

           Uses the Shipyard run_id as metadata tag to find the corresponding
           LangSmith trace, then shares it publicly.

           Returns the public URL or None if tracing is unavailable.
           """
       ```

       Implementation:
       - Import `os` and check `os.environ.get("LANGCHAIN_API_KEY")` — return None if not set
       - Import `langsmith.Client` inside the function (lazy import to avoid import errors when langsmith not installed)
       - Create `client = Client()`
       - Use `client.list_runs(project_name=os.environ.get("LANGCHAIN_PROJECT", "default"), filter=f'has(tags, "run_id:{run_id}")')` to find the LangSmith run by tag
       - If found, call `client.share_run(ls_run.id)` and return the URL
       - If not found by tag, try `client.list_runs(project_name=..., limit=5)` and match by timing/metadata
       - Wrap everything in try/except, return None on any error, log warning via logger
       - All langsmith Client calls are sync, so use `asyncio.get_event_loop().run_in_executor(None, ...)` to avoid blocking

    2. In `server/main.py`, modify the `execute()` inner functions (submit_instruction, continue_run, _resume_from_checkpoint) to:
       - Add `"tags": [f"run_id:{run_id}"]` to the config dict passed to `graph.ainvoke()` — this tags the LangSmith trace with the Shipyard run_id
       - After successful graph.ainvoke() completion (in the try block, after setting runs[run_id] status), call:
         ```python
         from agent.tracing import share_trace_link
         trace_url = await share_trace_link(run_id)
         if trace_url:
             runs[run_id]["trace_url"] = trace_url
         ```
       - Update the store run record with trace_url if available

