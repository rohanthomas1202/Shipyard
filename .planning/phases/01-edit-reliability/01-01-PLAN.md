Implement layered fuzzy anchor matching, indentation preservation, and file content hashing in the file operations layer.

Output: Enhanced `edit_file()` with fuzzy matching, `content_hash()` utility, `file_hashes` in AgentState, comprehensive tests.

    **1. Add imports to agent/tools/file_ops.py (top of file):**
    ```python
    import re
    import hashlib
    from difflib import SequenceMatcher
    ```

    **2. Add helper functions BEFORE edit_file():**

    `FUZZY_THRESHOLD = 0.85` module constant.

    `_normalize_whitespace(text: str) -> str` — `re.sub(r'\s+', ' ', text).strip()`

    `_find_fuzzy_match(content: str, anchor: str, threshold: float = FUZZY_THRESHOLD) -> tuple[str | None, float]` — Slide a window of `anchor_lines = anchor.count('\n') + 1` lines across `content.split('\n')`. For each candidate window, compute `SequenceMatcher(None, anchor, candidate).ratio()`. Track best. Return `(best_match, best_score)` if `best_score >= threshold`, else `(None, best_score)`.

    `_detect_indent(line: str) -> tuple[str, int]` — Strip leading whitespace. If `\t` in indent, return `('\t', count_tabs)`. Else return `(' ', len_spaces)`.

    `_adjust_replacement_indent(anchor_text: str, replacement: str, content: str) -> str` — Find anchor_text position in content. Get the context line's indent via `_detect_indent`. Get replacement's first non-empty line indent. If they match, return replacement unchanged. Otherwise re-indent each replacement line: `new_indent = char * (base_level + (line_level - rep_base))`, preserving relative indent. Empty/whitespace-only lines pass through unchanged.

    `content_hash(content: str) -> str` — `hashlib.sha256(content.encode()).hexdigest()[:16]`

    **3. Replace edit_file() with the layered version:**

    The new `edit_file(path, anchor, replacement)` function:
    - Reads file content (FileNotFoundError returns error dict as before)
    - **Layer 1 (exact):** `content.count(anchor)` — if 1, apply `_adjust_replacement_indent(anchor, replacement, content)`, write, return `{"success": True, "snapshot": content, "error": None, "match_type": "exact"}`; if >1, return not-unique error as before
    - **Layer 2 (whitespace-normalized):** Slide window of `anchor.count('\n') + 1` lines, compare `_normalize_whitespace(candidate) == _normalize_whitespace(anchor)`. If exactly one match found, apply `_adjust_replacement_indent`, write, return `{"success": True, ..., "match_type": "whitespace_normalized"}`
    - **Layer 3 (fuzzy):** Call `_find_fuzzy_match(content, anchor)`. If match found, apply `_adjust_replacement_indent(matched_text, replacement, content)`, write, return `{"success": True, ..., "match_type": "fuzzy", "fuzzy_score": score}`
    - **All failed:** Call `_find_fuzzy_match(content, anchor, threshold=0.0)` to get best score. Return `{"success": False, "snapshot": None, "error": f"Anchor not found in {path}", "best_score": best_score, "anchor_preview": anchor[:200], "best_match_preview": best_match[:200] if best_match else ""}`

    Note: For Layer 2, handle the case where multiple whitespace-normalized matches exist (return not-unique error).

    **4. Add file_hashes to AgentState in agent/state.py:**
    Add `file_hashes: dict[str, str]` field to the TypedDict. Place it after `file_buffer`.
