    1. Create `agent/orchestrator/scheduler.py` with `DAGScheduler` class (per D-02):
       - Module docstring: `"""Async DAG-aware task scheduler with dependency enforcement and crash recovery."""`
       - `__init__(self, dag: TaskDAG, dag_run: DAGRun, *, persistence: DAGPersistence | None = None, event_bus: EventBus | None = None, task_executor: Callable[[str, TaskNode], Awaitable[dict]] | None = None, max_concurrency: int = 10)`:
         - `self._dag = dag`
         - `self._dag_run = dag_run`
         - `self._persistence = persistence`
         - `self._event_bus = event_bus`
         - `self._task_executor = task_executor or self._default_executor`
         - `self._semaphore = asyncio.Semaphore(max_concurrency)`
         - `self._completed: set[str] = set()`
         - `self._failed: set[str] = set()`
         - `self._running: set[str] = set()`
         - `self._progress_event = asyncio.Event()` -- set when any task finishes, so scheduling loop wakes up

       - `async def run(self) -> dict[str, str]`: Main scheduling loop:
         1. If persistence exists, call `mark_interrupted(dag_id)` to handle Pitfall 3
         2. Load completed tasks from persistence into `self._completed`
         3. Emit `dag_started` event
         4. Loop: get ready tasks (not in completed/failed/running), start each with `asyncio.create_task(self._run_task(task_id))`, await `self._progress_event.wait()` then clear it
         5. Break when no ready tasks and no running tasks
         6. Emit `dag_completed` or `dag_failed` event
         7. Return `{task_id: status}` map

       - `async def _run_task(self, task_id: str) -> None`:
         1. `async with self._semaphore:` (concurrency gate)
         2. Emit `task_started` event, persist status="running"
         3. Call `self._task_executor(task_id, self._dag.get_task(task_id))`
         4. On success: persist status="completed", add to `self._completed`, emit `task_completed`
         5. On exception: persist status="failed", add to `self._failed`, emit `task_failed` with error
         6. Remove from `self._running`, set `self._progress_event`

       - `async def _default_executor(self, task_id: str, task: TaskNode) -> dict`: No-op executor that returns `{"success": True}`. Used in tests.

       - `async def _emit(self, event_type: str, task_id: str | None = None, data: dict | None = None) -> None`: Helper that creates Event and calls `event_bus.emit()` if event_bus is set.

       Key design: Use `asyncio.create_task()` for downstream scheduling (not await) to avoid Pitfall 2 (deadlock). The `_progress_event` wakes the main loop without polling.
